# Concepts

This directory stores product-level concept documents for `cc-bash-proxy`.

- Scope: problem statement, target users, value proposition, non-goals
- Out of scope: command contracts and implementation behavior under
  `docs/dev/spec/`

## Files

- `docs/concepts/product-concept.md`: core product concept and operator-facing
  value proposition
- `docs/concepts/security-trust-model.md`: trust boundary, supply-chain risk,
  and publication-time security stance
